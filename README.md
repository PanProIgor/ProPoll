Приложение "ProPoll" — это полнофункциональное веб-приложение для проведения опросов, созданное на фреймворке Django. Каждый зарегистрированный пользователь  может добавлять, редактировать, обновлять, удалять опросы, добавлять варианты ответов, обновлять варианты ответов, то же самое можно сделать через административный блок сайта.  Пользователь может просматривать опросы и голосовать. Если пользователь  проголосовал, то не сможет проголосовать повторно. Если опрос завершён, проголосовать за него нельзя. В завершённом опросе отображается только окончательный результат. Есть возможность поиска опросов. Также пользователь может фильтровать опросы по названию, дате публикации и количеству голосов.
